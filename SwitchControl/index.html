About this app

This app is a calculator that allows for switch controls

  • This app uses the WASD keys to navigate the calculator: W is for up, A is for left, S is for down, and D is for right. The F key is used to select a button
  • Test results: The calculator is very easy to use if you have ever used WASD controls before. If you have not, they are very similar to the arrow keys on a keyboard.
  • I also tried three button controls. J was used to go left, and if you cannot go left anymore it moves up and to the farthest right. K was used to go right until you cannot go right anymore, then down and to the farthest left. L was used to select the current button.
